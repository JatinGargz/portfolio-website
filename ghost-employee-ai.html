Ghost Employee AI

AI-Powered Workforce Analytics & Productivity Platform mapping unstructured work logs into visual analytics.

The Challenge

Auditing raw, unstructured employee activity updates, timesheets, and workspace records manually is time-consuming for manager reviews and fails to provide structured patterns or highlight operational bottlenecks. Unstructured text logs often miss standardized time expressions and consistent structural formats, making it hard to build visual timeline charts.

System Architecture

We developed a data pipeline that ingests raw log uploads and parses them using Google's Gemini models to generate clean, structured metrics.

Raw Logs

Summary Generator
Timeline Visualizer
Metrics Board

The React-based frontend allows managers to upload log files, which are processed through Node.js endpoints. The backend uses zero-shot schemas to prompt Gemini, returning strict JSON vectors containing parsed timestamps, task descriptions, and performance scores.

Measurable Impact

100+ Concurrent Activities Parsed

The platform reduces employee review times significantly, parsing over a hundred concurrent raw updates and outputting formatted performance reports under two minutes.

Challenges & Key Learnings

Log parsing complexity: Handling inconsistent activity log syntax and non-standard time expressions (e.g. "two hours in the morning", "10-12"). Resolved this by using few-shot prompt schemas in Gemini to force output formatting into uniform ISO-8601 time ranges, ensuring front-end charting compatibility.

Future Improvements

  • Integrate calendar APIs to auto-match meeting events with logs.
  • Add vector embeddings search to let managers ask conversational questions about team logs.

AI Content Platform

An automated content generation pipeline connecting brief prompts directly to custom visual models.

The Challenge

Creating marketing narratives, visual assets, and voiceovers is a highly manual process. Integrating different generation models (narrative, image, video, voice) often leads to visual inconsistencies, character faces breaking across slides, and significant delay in render queues.

System Architecture

We built a visual node-based generation manager linking LLMs with stable diffusion and audio templates.

Campaign Brief
Gemini Script
Stable Diffusion / ComfyUI
Video Synthesis
Ad Output

Users submit a brief, which Gemini processes into copy scripts. ComfyUI pipelines ingest the outputted JSON to render matching visual assets. The voice tracks are synthesized via ElevenLabs APIs and combined with generated video clips.

Project Metadata

Focus Multimodal Generative AI

Role Solo pipeline Engineer

Measurable Impact

80% Reduction in Manual Creation Time
100% Vocal & Visual Consistency

The platform automates the creation pipeline, allowing users to compile full campaigns within minutes rather than hours.

Challenges & Key Learnings

Character consistency: Image generation models often yield differing details on consecutive runs. We resolved this by integrating IP-Adapter control models inside ComfyUI, mapping a static facial vector to guide generation weights.
